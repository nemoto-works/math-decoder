# 算数デコーダー / Math Decoder

**見えない考え方を、見える形に。**

3桁÷2桁の筆算で、子どもが「いま、どこを見るのか」を理解できるようにするWeb MVPです。

## 公開方法

Vercelで公開します。

設定:

- Framework: `Vite`
- Build Command: `npm run build`
- Output Directory: `dist`

## MVPの範囲

- 題材は `156 ÷ 12` の1問固定
- 学校と同じ筆算形式で表示
- スポットライトで視線誘導
- 視線クイズで「どこを見るか」を確認
- 「見る → たてる → かける → ひく → おろす」を1ステップずつ表示
- 下段にブロック・グループ・合体表示のビジュアル補助
